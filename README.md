# 📰 Dynamic News Portal

A modern and responsive news web application that delivers the latest headlines across different categories including Sports, Technology, and Entertainment using a News API integration.

## 🚀 Live Demo

Demo Link:
https://apiworldnews.netlify.app/

🎥 Demo Video:

https://github.com/user-attachments/assets/41912274-199a-42cf-9857-2d1b7e0ae810

---

## 📌 Features

- Real-time news updates from News API
- Search functionality for finding specific news articles
- Category-based filtering
  - Sports
  - Technology
  - Entertainment
- Fully responsive design
- Clean and modern UI
- Direct links to original news sources
- Fast loading and lightweight application

## ⚙️ Installation

### Clone Repository

git clone https://github.com/Cod3shield/Global-News-Portal

### Open Project

cd dynamic-news-portal

Open `index.html` in your browser or run with VS Code Live Server.

---

## 🔑 API Setup

Get your API key from:

https://newsapi.org/

Replace the API key inside your JavaScript file:

javascript
const API_KEY = "YOUR_API_KEY";
